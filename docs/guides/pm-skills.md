# Product Management Skills Guide

Here are the **46 PM skills** organized by what they can do:

## Writing & Artifacts (Component Skills)

| Skill | What It Does |
|-------|-------------|
| **user-story** | Write stories with Mike Cohn format + Gherkin acceptance criteria |
| **epic-hypothesis** | Turn initiatives into testable hypotheses with success metrics |
| **problem-statement** | Frame customer problems with evidence before solutions |
| **positioning-statement** | Geoffrey Moore-style positioning (who, what, how different) |
| **press-release** | Amazon Working Backwards - write future PR first |
| **prd-development** | Full PRD: problem → personas → solution → metrics → stories |
| **jobs-to-be-done** | Understand what customers are trying to accomplish (JTBD) |
| **customer-journey-map** | Map experience across all touchpoints (NNGroup) |
| **user-story-mapping** | Organize stories by user workflow (Jeff Patton) |
| **user-story-splitting** | Break large stories using 8 proven patterns |
| **storyboard** | 6-frame visual narrative of user journey |
| **proto-persona** | Create hypothesis-driven personas before research |
| **company-research** | Deep-dive competitor or company analysis |
| **pestel-analysis** | Analyze external factors (Political, Economic, Social, Tech, Env, Legal) |
| **pol-probe** | Define lightweight validation experiments before building |
| **eol-message** | Communicate product/feature deprecation gracefully |
| **recommendation-canvas** | Document AI-powered product recommendations |

## Decision-Making & Strategy (Interactive Skills)

| Skill | What It Does |
|-------|-------------|
| **prioritization-advisor** | Recommend RICE, ICE, Kano, etc. based on your situation |
| **discovery-process** | Full discovery cycle: problem → research → synthesize → validate |
| **product-strategy-session** | Positioning → problem framing → solution → roadmap (2-4 weeks) |
| **roadmap-planning** | Gather inputs → epics → prioritize → sequence → communicate |
| **opportunity-solution-tree** | Generate opportunities/solutions, recommend best POC |
| **problem-framing-canvas** | MITRE Problem Framing (Look Inward/Outward/Reframe) |
| **lean-ux-canvas** | Hypothesis-driven planning (Jeff Gothelf) |
| **positioning-workshop** | Define positioning with adaptive questions |
| **discovery-interview-prep** | Plan customer interviews (Mom Test style) |
| **customer-journey-mapping-workshop** | Guided journey mapping with pain points |
| **user-story-mapping-workshop** | Create story maps with backbone + release slices |
| **workshop-facilitation** | One-step-at-a-time facilitation for workshops |

## Analysis & Metrics (Interactive Skills)

| Skill | What It Does |
|-------|-------------|
| **business-health-diagnostic** | Diagnose SaaS health: growth, retention, efficiency, capital |
| **saas-revenue-growth-metrics** | ARPU, MRR/ARR, churn, NRR, expansion metrics |
| **saas-economics-efficiency-metrics** | CAC, LTV, payback, margins, burn rate, Rule of 40 |
| **tam-sam-som-calculator** | Project market size with real-world data |
| **finance-metrics-quickref** | Fast lookup: 32+ SaaS metrics with formulas |
| **finance-based-pricing-advisor** | Evaluate pricing changes (ARPU, conversion, churn risk) |
| **feature-investment-advisor** | Build/don't build recommendations via ROI + strategy |
| **acquisition-channel-advisor** | Evaluate channels: scale/test/kill decisions |
| **epic-breakdown-advisor** | Split epics using Richard Lawrence's 9 patterns |
| **pol-probe-advisor** | Recommend prototype type based on hypothesis + risk |
| **ai-shaped-readiness-advisor** | Assess AI-first vs AI-shaped product maturity |
| **context-engineering-advisor** | Context stuffing vs context engineering diagnosis |

## Usage

Just describe what you need - the skill will auto-activate based on context:

- "Help me prioritize 12 backlog items" → `prioritization-advisor`
- "Write a user story for checkout" → `user-story`
- "Run a product strategy session" → `product-strategy-session`
- "What's our market size?" → `tam-sam-som-calculator`
