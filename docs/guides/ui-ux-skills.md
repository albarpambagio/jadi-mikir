# UI/UX Pro Max Skills Guide

Comprehensive design intelligence skill with **67 UI styles**, **161 color palettes**, **57 font pairings**, **99 UX guidelines**, and **25 chart types** across **13 technology stacks**.

## Quick Start

### Basic Workflow

```bash
# Step 1: Generate Design System (REQUIRED)
python3 skills/ui-ux-pro-max/scripts/search.py "beauty spa wellness" --design-system -p "My Project"

# Step 2: Supplement with detailed searches (optional)
python3 skills/ui-ux-pro-max/scripts/search.py "glassmorphism" --domain style

# Step 3: Get stack-specific guidelines
python3 skills/ui-ux-pro-max/scripts/search.py "form validation" --stack react
```

### Prerequisites

```bash
# Check Python
python3 --version || python --version

# Install Python if needed
# macOS: brew install python3
# Ubuntu: sudo apt install python3
# Windows: winget install Python.Python.3.12
```

## Design System Command

### Generate with ASCII Output

```bash
python3 skills/ui-ux-pro-max/scripts/search.py "fintech banking" --design-system -p "MyApp"
```

### Generate with Markdown Output

```bash
python3 skills/ui-ux-pro-max/scripts/search.py "fintech banking" --design-system -f markdown
```

### Persist Design System (Master + Overrides)

```bash
# Create MASTER.md
python3 skills/ui-ux-pro-max/scripts/search.py "saas dashboard" --design-system --persist -p "MyApp"

# Create page-specific override
python3 skills/ui-ux-pro-max/scripts/search.py "saas dashboard" --design-system --persist -p "MyApp" --page "checkout"
```

**Folder structure created:**
```
design-system/
├── MASTER.md           # Global design rules
└── pages/
    └── checkout.md     # Page-specific overrides
```
