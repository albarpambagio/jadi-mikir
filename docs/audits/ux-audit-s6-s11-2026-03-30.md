# UX Audit: Progress (S6), Session complete (S11), session path (S2–S3)

**Date**: 2026-03-30  
**Scope**: [`src/pages/progress.tsx`](../../src/pages/progress.tsx), [`src/components/session/session-complete-view.tsx`](../../src/components/session/session-complete-view.tsx), session flow in [`src/pages/session.tsx`](../../src/pages/session.tsx).  
**Method**: Code review + flow checklist against [`docs/archives/wireframes_1.md`](../../docs/archives/wireframes_1.md) S2, S3, S6, S11. **Confidence**: Static review — validate in browser (`npm run dev`, `http://localhost:5173`).

## Persona

**Dewi**, 15, Indonesian SMP learner — wants to see overdue work quickly, finish a review session, and understand results without English friction.

## Flow scores

| Flow | Task completed? | Clicks to goal | Confusion / anxiety | Notes |
|------|-------------------|----------------|---------------------|--------|
| S6 Progress | Yes | 1–2 to session from alert | **Medium** when UI mixes EN page chrome with ID alert | Top alert matches wireframe intent |
| S2–S3 Session | Yes | 2+ per item (select → confirm → next) | **Low** on happy path | Quit-only exit is clear; feedback inline is scannable |
| S11 Complete | Yes | 1 to home or continue | **Medium** if band/weak area still English vs ID schedule | Section order matches wireframe |

## Findings

### High

1. **Language inconsistency (S6 + S11 + session)**  
   *Where*: Progress title/back/summary vs Indonesian overdue block; session `Question`, `Confirm answer`, `Quit session`, difficulty `Easy`; session complete English bands, stats labels, weak-area copy, `Back to home`.  
   *Issue*: Same flows mix ID and EN — raises cognitive load and undermines wireframe Indonesian examples.  
   *Fix*: Align learner-facing strings to Indonesian (implemented in copy pass + code).

2. **Session complete: weak-area card still English**  
   *Where*: `SessionCompleteView` — "You missed…", "Practice this area now", "Remind me next session".  
   *Issue*: Juxtaposes with Indonesian *Jadwal berikutnya*.  
   *Fix*: Indonesian strings (implemented).

### Medium

3. **S6: Duplicate overdue information**  
   *Where*: Top alert + retention row "Cards currently overdue".  
   *Issue*: Not wrong, but verbose when both visible.  
   *Fix (optional)*: Shorten retention line when `totalDue > 0` or rely on top card only — deferred; copy doc notes.

4. **S11: `formatSessionDuration` English**  
   *Where*: Time stat uses `< 1 min` / `N min`.  
   *Issue*: Rest of summary moves Indonesian.  
   *Fix*: `formatSessionDurationId` for session complete (implemented).

5. **Accessibility: session complete `h1` vs eyebrow**  
   *Where*: Eyebrow "Session complete" / `h1` topic or "Your results".  
   *Issue*: Already improved in prior audit; still verify screen reader order after copy change.  
   *Fix*: No code change this pass beyond Indonesian eyebrow.

### Low

6. **Narrow viewport: S6 alert row**  
   *Where*: Flex wrap on overdue card — CTA may drop below text.  
   *Issue*: Acceptable; test 375px when possible.

7. **Difficulty labels**  
   *Where*: Mudah/Sedang/Sulit vs content language — assumes UI locale ID.
